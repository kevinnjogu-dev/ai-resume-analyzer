# AI Resume Analyzer

An AI-powered resume analysis toolkit that evaluates resumes against job descriptions using structured scoring, prompt engineering concepts, and automated quality analysis.

This project demonstrates practical workflows used in **LLM evaluation, prompt engineering, resume screening, and AI-assisted candidate assessment**.

---

## Project Overview

Recruiters and hiring teams often review hundreds of resumes for a single role. This toolkit simulates an AI-powered resume evaluation workflow by comparing resumes against job descriptions and generating structured evaluation scores.

The analyzer evaluates resumes across multiple dimensions:

- Skills Match
- Experience Relevance
- Education Alignment
- Keyword Coverage
- Resume Clarity
- Overall Match Score

---

## Features

- Resume and job description comparison
- Structured resume scoring
- Keyword matching
- JSON-based evaluation results
- Modular Python architecture
- Visualization of evaluation scores
- Unit testing with pytest
